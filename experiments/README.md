# Experiments

Reproduces the TokLens evaluation across 24 tokenizers and 15 languages.

## Pipeline

Run steps in order:

```bash
cd toklens
uv run python -m experiments.01_collect_benchmarks
uv run python -m experiments.02_compute_metrics
uv run python -m experiments.03_correlation
uv run python -m experiments.04_figures
```

Or all at once:

```bash
bash experiments/run_all.sh
```

## Data Sources

- Benchmark scores: [Open LLM Leaderboard v2](https://huggingface.co/datasets/open-llm-leaderboard/contents)
- Multilingual corpora: [Wikipedia](https://huggingface.co/datasets/wikimedia/wikipedia) (100 articles per language, ~50K chars)
- Tokenizers: loaded from HuggingFace Hub (non-gated mirrors for gated models)

## Configuration

All tokenizers, languages, and benchmark columns are defined in `config.py`.

## Key Implementation Notes

- **Special tokens**: All tokenizer encoding uses `add_special_tokens=False` for fair comparison. Without this, tokenizers like Llama and Gemma add BOS tokens that inflate metrics.
- **Gated models**: Llama-2, Llama-3.1, and Gemma-2 are loaded from non-gated mirrors (NousResearch, unsloth). Command-R uses Xenova's tokenizer-only repo.
- **Sentencepiece models**: GLM-4 uses tiktoken and requires a slow tokenizer shim via `transformers.AutoTokenizer`.
- **Shared tokenizers**: Some models share tokenizers (e.g., Llama-2 = Phi-3-mini, OLMo-2 = Phi-4 = StableLM-2). Qwen2.5 family uses the same tokenizer across all sizes.
